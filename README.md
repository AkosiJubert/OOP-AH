## Project Overview

The MotorPH Payroll System is a progressive implementation developed across three programming courses:

- **Computer Programming 1** – Console-based payroll system  
- **Computer Programming 2** – GUI-based payroll system  
- **Object-Oriented Programming** – Refactored system applying Object-Oriented principles such as Encapsulation, Abstraction, Inheritance, and Polymorphism  

This version represents the complete OOP implementation with proper backend logic and GUI integration.

---

## About MotorPH

MotorPH was established in 2020 to provide affordable and competitive motorcycles to Filipinos. As the company expands into physical branches nationwide, a scalable payroll system is essential to support employee and financial management.
